# HackPad

This is a 12 key macropad with standard mx switchs and one LED per key

## Features
12 keys
16 LEDs
Keyboard Matrix

## CAD Model

Uses m3 bolts and heat set inserts to attach the PCB to the acual case.
3 parts of the case, seprely printed, the bottom, the PCB protector and the lid.

## BOM
- 12 Cherry MX Switches
- 12 1U DSA Keycaps
- 12 1N4148 DO-35 Diodes
- 7 M3x5x4 Heatset inserts
- 16 SK6812 MINI-E LEDs
- 1x XIAO RP2040
- 3D printed parts
- A PCB
